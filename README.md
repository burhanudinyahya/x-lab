## Tools dan Framework
* Apache 2.4.25
* MariaDB 10.1.23
* PHP 7.2.1
* Boostrap 3.3.6

### Clone repo

```
> git clone git@github.com:burhanudinyahya/x-lab.git
> cd x-lab
> code .
```

Create database with name 'xlab' and execute file .sql (file in db folder) for create table and dummy data.
